PrimeHill Docs 1.0 Help

Сценарии использования

Списание бонусов

  1. Создать заказ

  2. Добавить гостя в заказ

  3. Перейти на экран оплаты

  4. Добавить тип оплаты бонусами

  5. Нажать "Да"

  6. Оплатить заказ

Не обязательно списывать все доступые бонусы. После добавления платежа сумму списания всегда можно изменить. Однако максимальное количество бонусов фиксировано и больше этого числа списать нельзя

Списание депозита

  1. Создать заказ

  2. Добавить гостя в заказ

  3. Перейти на экран оплаты

  4. Добавить тип оплаты депозитом

  5. Ввести номер депозитной карты

  6. Выбрать тип списания

  7. Оплатить заказ

Пополнение депозита

Настройка

  1. Создать заказ

  2. Добавить в заказ товар, созданный для пополнения (инфа о настройке)

  3. Перейти на экран оплаты

  4. Добавить тип оплаты для пополнения

  5. Ввести номер карты

  6. Оплатить заказ

Для пополнения депозита также можно использовать доставочные заказы. Логика такая же. Добавить в заказ товар для пополнения и оплатить типом для пополнения

Использование скидок, рассчитанных в PrimeHill

Настройка

  1. Создать заказ

  2. Добавить гостя в заказ

  3. Перейти на экран оплаты

  4. Скидка автоматически добавиться как платеж

  5. Оплатить заказ

Использование скидок, рассчитанных в iiko

Настройка

Карта гостя должна значиться как скидочная карта в iikoOffice

  1. Создать заказ

  2. Добавить гостя в заказ

  3. К заказу автоматически примениться скидка

  4. Перейти на экран оплаты

  5. Оплатить заказ

Импорт гостей

Плагин в процессе работы через равные промежутки времени запрашивает новых гостей с сервера, чтобы синхронизировать гостей в PrimeHill и в базе iiko

    Получение информации о госте

    1. Нажать кнопку "Дополнения"

    2. Нажать "PrimeHill: Информация о госте"

    3. Ввести номер карты или телефон гостя

    4. Будет отображена информация о госте

    5. При клике на кнопку "Обновить гостя" информация о госте в БРД будет обновлена в соответствии с информацией из PrimeHill

    Отправка логов

    1. Нажать кнопку "Дополнения"

    2. Нажать "PrimeHill: Отправить логи"

    3. Выбрать дату, за которую отправляются логи

    4. Логи будут отправлены на сервер. Найти их можно в личном кабинете PrimeHill в Справочники -> Файлы

    Чекин гостя с экрана кассы

    Заказ не в статусе пречека

    1. Создать заказ

    2. Перейти на экран кассы

    3. Добавить тип оплаты PrimeHill

    4. Ввести номер карты/телефон

    5. Продолжить по сценарию списания бонусов

      или скидки PrimeHill

    Заказ в статусе пречека

    1. Создать заказа

    2. Напечатать пречек

    3. Перейти на экран оплаты

    4. Добавить тип олаты PrimeHill

    5. Ввести пин-код администратора, заданный в настройках плагина

    6. Продолжить по сценарию чекина с экрана кассы

    Конвертация скидки в бонусы для доставочных заказов

    1. Создать доставку через call-центр или любой другой сервис

    2. Добавить скидку, прописанную в конфиге и сохранить на точку

    3. Дождаться появления доставки на iikoFront и отправить ее

    4. Сумма скидки станет платежем PrimeHill

    Работа с 2ФР

    Ссылка на настройку 2ФР

    1. Создать заказ

    2. Добавить в него блюда с разным типом места приготовления

    3. Напечатать пречек

    4. Перейти на экран кассы, разделив заказ

    5. Доступные для оплаты бонусы будут разделены между двумя созданными заказами

    Комбинирование оплаты бонусами и депозитом

    Бонусы и депозит на разных картах

    1. Добавить бонусный платеж, согласно сценарию списания бонусов

    2. Добавить платеж депозитом, согласно сценарию списания депозита

    Бонусы и депозит на одной карте

    Агрессивный экспорт

    1. Подложить на кассу EXE-файл активатора "агрессивного экспорта"

    2. Проверить версию плагина PrimeHill (версия должна быть 7.10.0 и выше)

    3. Запустить кассу

    4. Запустить активатор, убедиться во всплывающем уведомлении на кассе, что экспорт включен

      Если у вас плагин работает не на стандартном порту (8115), то активатор не сможет сразу включить режим "агрессивного экспорта" о чем вам и сообщит, после чего спросит, не хотели бы вы указать другой порт. Отвечаем да и вводим тот порт, на котором работает плагин. После этого режим должен успешно запуститься

    5. Перейти в личный кабинет PrimeHill

    6. Выбрать нужных гостей и нажать "экспорт"

    7. Дождаться завершения работы "агрессивного экспорта" (плагин пришлет уведомление на кассу)

    8. Проверить в папке кеша плагина файл warnings.txt. В этом файле сохраняется информация о тех гостях, которых не получилось экспортировать по тем или иным причинам (например, было найдено несколько гостей с одинаковым номером телефона или не получилось изменить номер карты гостя, потому что этот номер карты уже привязан к другому гостю)
